# 3 Idiots — Audience Review Statistical Analysis

A descriptive statistical analysis of 1,113 real IMDb audience reviews 
for the Bollywood film *3 Idiots* (2009), built as part of a statistics 
and Python project.

## Key Findings
- **Mean Rating:** 8.52 / 10
- **Median:** 9.0 | **Mode:** 10.0
- **Skewness:** −1.853 (negatively skewed — audience loved it)
- **Standard Deviation:** 2.15 (high consensus)
- **85%** of reviews were Positive

## Tech Stack
Python · pandas · numpy · matplotlib · seaborn · scipy · wordcloud

## How to Run
1. Clone the repo
2. Place `3idiots_reviews.csv` in the root folder
3. Open `3idiots_audience_analysis.ipynb` in Jupyter
4. Run all cells

## Project Structure
- `3idiots_audience_analysis.ipynb` — main analysis notebook
- `3idiots_reviews.csv` — dataset (1,113 IMDb reviews)
- `outputs/` — all generated charts and graphs
